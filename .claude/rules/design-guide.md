# Design Guide

UI/UX guidance for the web application.

## Principles
- Prioritize clarity and hierarchy.
- Use consistent spacing, typography, and states.
- Provide immediate feedback for user actions.

## Responsiveness
Use Tailwind breakpoints:
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
- `2xl`: 1536px

## Accessibility
- Use semantic elements where possible.
- Ensure keyboard access for interactive UI.
- Provide visible focus states and adequate contrast.
